# famcuaminha

## Lệnh tạo ảnh + chèn trực tiếp logo (không sửa code)

Dùng prompt sau trong ChatGPT/DALL·E/Midjourney để tạo banner có **logo đính kèm trực tiếp**:

```bash
/imagine prompt: Thiết kế ảnh quảng bá dịch vụ số cao cấp cho thương hiệu "FAM CỦA MINH A", phong cách neon xanh tím, nền sạch, bố cục tròn hiện đại. GIỮ NGUYÊN logo tham chiếu, đặt logo ở trung tâm, không vẽ lại chữ trong logo, không đổi màu logo gốc. Thêm các nhãn dịch vụ nhỏ: ChatGPT, YouTube, Netflix, CapCut, Canva. Tỉ lệ 1:1, chất lượng cao, sắc nét, phù hợp avatar và ảnh bìa.
```

### Cách chèn trực tiếp logo bạn đã gửi (không code)

1. Mở Canva.
2. Upload logo gốc bạn đã gửi.
3. Đặt logo vào chính giữa thiết kế.
4. Xuất file PNG (nền trắng hoặc trong suốt tùy nhu cầu).

> Mẹo: bật tùy chọn “lock element” cho logo trong Canva để tránh xê dịch khi chỉnh bố cục.
